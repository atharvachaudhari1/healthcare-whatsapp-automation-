## ✨ Overview

DigiGrowNex is a complete WhatsApp CRM and Automation Platform that helps businesses manage customer conversations, automate workflows, track leads, run campaigns, and monitor performance from a single dashboard.

---

## 🔥 Features

### 💬 Smart Shared Inbox

* Multi-agent collaboration
* Conversation assignment
* Internal notes
* Real-time messaging

### 👥 Contact Management

* Customer database
* Tags & segmentation
* Custom fields
* Import & export support

### 📈 Sales Pipeline

* Kanban-style deal tracking
* Lead lifecycle management
* Opportunity monitoring

### 📢 Broadcast Campaigns

* Bulk WhatsApp campaigns
* Template messaging
* Delivery & read tracking

### 🤖 Automation Engine

* Keyword triggers
* Scheduled workflows
* Conditional logic
* Webhook integrations

### 📊 Analytics Dashboard

* Response metrics
* Campaign insights
* Team performance
* Growth monitoring

## 🚀 Quick Start

```bash
git clone https://github.com/atharvachaudhari1/DigiGrowNex.git

cd DigiGrowNex

npm install

npm run dev
```

Open:

```bash
http://localhost:3000
```
